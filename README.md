# Proxy
> HTTP請求代理，CORS跨域請求，HTTPS支持

### 作用
- 支持跨域請求（轉換不支持跨域請求的接口），可直接發起 ajax、fetch
- 支持HTTPS（解決遠程數據接口不支持HTTPS）

### 貢獻
- Heroku
